# GDPR Principles vs WP29 Criteria Matrix

This repository provides a lightweight framework that maps core GDPR principles to WP29 DPIA criteria in the context of serious game validation studies.

The goal is to support researchers in understanding how high-level GDPR principles (e.g., transparency, fairness, data minimisation, and accountability) relate to WP29 risk indicators such as profiling, systematic monitoring, sensitive data processing, and large-scale data collection.

The framework is intended as a decision-support tool to help identify potential data protection risks and DPIA-relevant scenarios during the design and validation of serious games.

This material has been developed as part of research prepared for the _Simposio Internacional de Informática Educativa (SIIE 2026)_.
